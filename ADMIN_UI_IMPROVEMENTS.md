# Admin UI Improvements - Simpler & More Functional

## Current Issues
1. ❌ Too complex modal-based editing
2. ❌ No visual preview of customer experience
3. ❌ Rules management not intuitive
4. ❌ Missing version management UI
5. ❌ No bulk operations
6. ❌ Benefits editing is clunky

## Proposed Improvements

### 1. Simplified Combo Tiers List
- Card-based view (like customer-facing cards)
- Quick actions: Edit, Duplicate, Activate/Deactivate
- Drag-and-drop for sort order
- Visual status indicators

### 2. Enhanced Detail Page
- Better tab navigation
- Inline editing where possible
- Live preview updates
- Rules builder (visual, not JSON)

### 3. Quick Actions
- Bulk activate/deactivate
- Copy tier
- Export/Import config

### 4. Better Rules Management
- Visual rule builder
- Pre-defined rule templates
- Test rules before saving
